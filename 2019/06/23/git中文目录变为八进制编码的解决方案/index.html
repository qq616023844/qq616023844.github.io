git中文目录变为八进制编码的解决方案

Posted on 2019-06-23 | Edited on 2019-06-25 | In BUG记录

现象:原本中文编码变成了/xxx/xxx之类的八进制码

解决:修改本地git仓库下的.git隐藏文件夹下config文件,在[core]部分新增quotepath = false字段保存即可
